Plastic Pollution Predictor

For this project, we wanted to examine the amount of plastic pollution that exists in our oceans, and be able to predict how large of an issue it will be in the future. To try and answer this, we worked to develop a web application that allows you to input values for a theoretical country to see if the waste contribution would be a problem for the future.
